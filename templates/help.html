Help Center

Welcome to Music Library

Music Library lets users browse releases, view album details, and organize music into collections. This version uses mock data for the frontend, but the layout is designed to connect to the database later.

Browsing Releases

Use the Browse page to view all available music releases. Each release card shows the title, artist, genre, format, label, and release date. Click View to open the full release page.

Release Details

The release detail page shows more information about a selected release, including its description, metadata, and track listing.

Collections

Collections group releases together, such as favorite albums, road trip music, or vinyl records. Collection pages show the releases included and basic stats like total tracks, formats, and genres.

Account Features

Users can register, log in, and log out. Account information is connected to PostgreSQL.

Need Help?

If something does not load, try returning to the Home page or logging in again.
